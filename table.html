Tables in HTML

Table ->

Arrangement of data in the form of rows and columns.
There are diffrent tags for table. 

1.table tag:- used to create a new table. 
2.tr tag:- used to add rows. 
3.td tag:- used to add data in a row. 
4.thead tag:- used to add head in table. 
5.tfooter tag:- used to add a footer in table. 
6.tbody tag:- used to create body of table. 
7.th tag:- used to write in head.



Some commonly used attributes in table.

1.border -> to create border. 
2.rowspan -> to extend row. 
3.colspan -> to extend column.
