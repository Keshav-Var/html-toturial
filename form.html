Forms in Html

Form ->

The form HTML element represents a document section containing interactive controls for submitting information.

Some common form elements.

1.input -> for taking different type of inputs.

2.lable -> to add lable on input.

Note:-the for of label must be same as id of input.

3.select -> to create drop down menu.

4.option -> to create drop down menu options.

5.button -> to create button.

6.textarea -> to take multiline input.

7.fieldset -> grouping of elements.

8.legend -> adding caption.

*Types of input :

1.textfield ->to take multiline input.
2.button -> to create a button. set type = submit
3.radio -> to create radio button.
4.checkbox -> to create checkbox.
5.fileslector -> to create a file selector.
6.image -> to create image button.
7.password -> to create password box.
8.date -> to create a datepicker.
